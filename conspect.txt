Для работы с Prisma ORM нужно:

1. Скачать библиотеку prisma      |npm i prisma|
2. Инициализируем prisma в проект |npx prisma init|
3. В файле .env указать актуальный путь к базе данных(При использовании SQLite указать путь к файлу)

Prisma client позволяет нам общаться с бд - CRUD - create/read/update/delete 
Datasource db - данные о бд:
provider - Тип бд(SQLite,Mysql,postgresql...)
url - ссылка к бд
для создания моделей используется ключевое слово model и с большой буквы название модели
каждая модель обязана иметь уникальный идентификатор что бы обозначить уникалтный идентификатор используется атрибут @id
атрибуты расширяют функционал полей и меняют их поведение и все атрибуты начинаются с символа "@"
атрибут @default задает значение по умолчанию, и функция autoincrement самостоятельно инкрементирует значения
типы для полей в модели обязаталельно указываются (Int, String, Boolean)
для указывания опциональных(необязательных) полей используется знак "?" после типа

для проведения миграций используется команда npx prisma migrate dev
seed - это процесс добавления тестовых данных или первоначальных данных
